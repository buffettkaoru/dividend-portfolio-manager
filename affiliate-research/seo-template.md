## 2. 記事タイトルのテンプレート（クリック率↑）

### 鉄則：32文字以内・数字・感情ワード・年号

**良いタイトル例**
- 「**摂食障害のカウンセリング｜15年苦しんだ私が選んだ3社【2026年】**」（29文字）
- 「**過食嘔吐をやめられた理由｜オンライン相談で変わった7つのこと**」（30文字）
- 「**摂食障害の病院、どう選ぶ？経験者が語る失敗しない3つの基準**」（30文字）
- 「**【体験談】自宅で受けられるメンタル診療で救われた話**」（26文字）
- 「**保険診療と自費カウンセリング、どちらを選ぶべきか【比較】**」（28文字）

### 避けるべきタイトル
- 「摂食障害について」← 曖昧すぎる
- 「治し方まとめ」← 断定的（薬機法リスク）
- 「絶対に治る方法」← Google も読者も信用しない

```
摂食障害経験者の視点でまとめました。
```

## 6. 構造化データ（Schema.org）

記事ページの `<head>` に追加すると、Google検索で**リッチリザルト**（星評価・FAQ表示）が出やすくなる。

### 体験談記事用（Article + Person）

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "記事タイトルをここに",
  "description": "メタディスクリプションをここに",
  "author": {
    "@type": "Person",
    "name": "運営者名",
    "description": "摂食障害を15年経験し、現在は回復過程にあります。",
    "url": "https://www.sesshokushougai.net/about"
  },
  "publisher": {
    "@type": "Organization",
    "name": "摂食障害.net"
  },
  "datePublished": "2026-04-22",
  "dateModified": "2026-04-22",
  "image": "https://www.sesshokushougai.net/ogp.jpg"
}
</script>
```

```html
<script type="application/ld+json">
{
```

## 7. YMYL（Your Money or Your Life）対策

摂食障害は医療領域 = Googleが最も厳しく評価するカテゴリ。以下を**全ページ**に実装。

### チェックリスト

- [ ] 運営者プロフィールページ（本名 or 匿名でも経歴詳細）
- [ ] 全記事の末尾に執筆者情報（経験年数・回復状況）
- [ ] プライバシーポリシー・免責事項の整備
- [ ] 「この記事は医療行為を推奨するものではありません」の注記
- [ ] 緊急相談窓口へのリンク（全ページ共通）
- [ ] 記事更新日の明記
- [ ] 医師監修がなくても「当事者視点」と明記
- [ ] ソース引用（学術論文・公的機関ページ）

### 記事末尾の必須テンプレート

```html
<div class="author-box">
```

# アフィリエイト × SNS リサーチ

摂食障害サイト（sesshokushougai.net）向けのアフィリエイトバナーとSEO対策テンプレート集。

## ファイル構成

| ファイル | 用途 |
|---|---|
| `banners.html` | WordPress貼付け用バナー5種（カウンセリング／オンライン診療／人間ドック／緊急窓口／比較表） |
| `seo-template.md` | キーワード・タイトル・記事構造・構造化データ・YMYL対策の完全テンプレート |

## 使い方

### バナー
1. `banners.html` をブラウザで開いてプレビュー確認
2. 使いたいバナーの `<pre>` 内のHTMLをコピー
3. WordPress記事編集画面で「カスタムHTML」ブロックを追加して貼り付け
4. `href="#"` を各ASPで取得したアフィリエイトURLに置換

### SEO
1. `seo-template.md` の「最優先で作るべき記事TOP10」から1本選ぶ
2. 該当のキーワード・記事構造テンプレートに沿って執筆
3. 記事内に適切なバナーを1〜3箇所配置
4. 記事末尾に執筆者情報＋緊急窓口バナー

## 推奨ASP

- **A8.net**（最優先・案件網羅）
- **アクセストレード**（オンライン診療系に強い）
- **afb**（メンタル・美容医療の単価高め）
- **バリューコマース**（EPARK系）
